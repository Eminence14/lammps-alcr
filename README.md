# LAMMPS Simulation: Tensile Deformation of Al-Cr
This project contains a LAMMPS script to model a uniaxial tensile test on an Aluminium-Chromium (Al-Cr) alloy.

#### Simulation: Tensile deformation (tensile test)
#### Material: Aluminium-Chromium (Al-Cr) random alloy
#### Potential: MEAM (library.meam & CrAl.meam)
#### Key Outputs: Stress-strain data (CSV) and atomic dump files for visualization.
